# Closing the Gap: Open Deep Search Brings State-of-the-Art AI Search to the Open Source World

The field of search augmented by large language models (LLMs) is rapidly evolving, offering powerful new ways to find and synthesize information. However, much of the cutting-edge performance has been concentrated in proprietary solutions like Perplexity's Sonar Reasoning Pro and OpenAI's GPT-4o Search Preview. This dominance by closed-source systems raises concerns about transparency, limits innovation, and creates barriers for developers.

A new paper introduces **Open Deep Search (ODS)**, an open-source framework aiming to democratize advanced search AI. ODS demonstrates performance comparable to, and in some cases exceeding, state-of-the-art proprietary systems on established benchmarks, all while being plug-and-play with open-source base LLMs.

The core idea behind ODS is to enhance the capabilities of existing open-source LLMs by integrating sophisticated web search tools and intelligent reasoning agents.

ODS is built with two main components that work in synergy with a user-selected base LLM:

1.  **Open Search Tool (OST):** A novel web search tool designed to provide high-quality, relevant context from the internet.
2.  **Open Reasoning Agent (ORA):** An agent that interprets the user's query and orchestrates a sequence of actions, including calls to tools like the OST, to arrive at an answer.

**The Open Search Tool: Getting Better Context**

Proprietary search AI solutions often rely on their own internal search infrastructures, which are opaque. Existing open-source alternatives typically pass raw Search Engine Result Page (SERP) snippets to the LLM. The Open Search Tool in ODS goes beyond this. It employs a more sophisticated process:

*   **Query Rephrasing:** Generates multiple variations of the user's query to improve coverage and diversity of results.
*   **Enhanced Retrieval & Augmentation:** It scrapes content from retrieved links, chunks passages, and re-ranks them based on relevance, ensuring that the LLM receives comprehensive and pertinent information. It also includes custom handling for popular sites like Wikipedia and ArXiv.

This focus on providing a high-quality, curated context to the LLM is critical for the performance of ODS.

**The Open Reasoning Agent: Thinking and Acting**

The Open Reasoning Agent is where ODS orchestrates the process of answering the query. The agent interprets the user's request and decides which tools to use and in what sequence. ODS offers two versions of this agent:

*   **ODS-v1 (ReAct-based):** Leverages the ReAct (Reasoning and Action) framework, which interleaves thought steps with actions (like using the search tool or a calculator). This version combines Chain-of-Thought (CoT) reasoning, self-consistency decoding, and dynamic few-shot prompting derived from a community campaign to guide the agent's behavior.
*   **ODS-v2 (CodeAct-based):** Based on the CodeAct approach, which uses executable code (specifically Python in this case, leveraging the SmolAgents framework) to represent actions and orchestrate tool calls. This aligns well with Chain-of-Code (CoC) reasoning and has shown effectiveness in complex tasks.

These agents can call the Open Search Tool for web information, a calculator (integrated via Wolfram Alpha) for numerical tasks, and even use the base LLM itself for continued reasoning.

**Benchmarking Against the Best**

The paper evaluates ODS on two popular factuality benchmarks: SimpleQA and FRAMES. SimpleQA tests factual accuracy on short-form questions, while FRAMES presents challenging multi-hop questions requiring the integration of information from multiple sources.

The results, particularly with the powerful open-source DeepSeek-R1 LLM as the base model, are impressive. ODS-v2+DeepSeek-R1 achieves 88.3% accuracy on SimpleQA and 75.3% on FRAMES.

Comparing ODS-v2+DeepSeek-R1 to key proprietary baselines:

*   **GPT-4o Search Preview:** ODS-v2 is slightly behind on SimpleQA (88.3% vs 90.0%) but significantly outperforms it on FRAMES (75.3% vs 65.6%).
*   **Perplexity Sonar Reasoning Pro:** ODS-v2 shows strong improvements on both benchmarks (88.3% vs 85.8% on SimpleQA and 75.3% vs 44.4% on FRAMES).

The results demonstrate that ODS, by combining a strong open-source reasoning LLM with its specialized Open Search Tool and intelligent reasoning agent, can reach and surpass the performance of leading closed-source search AI solutions.

**The Promise of Open Source**

Achieving state-of-the-art performance in search AI with an open-source framework like ODS is a significant step towards democratizing this technology. By providing a plug-and-play system compatible with various LLMs, ODS empowers researchers and developers to build upon and innovate in the search AI space without relying on proprietary black boxes. The public release of the ODS implementation invites community collaboration, fostering transparency and accelerating progress in the field.

ODS represents a compelling open alternative, demonstrating that advanced search AI capabilities can be built and shared openly, narrowing the gap with closed-source leaders and paving the way for future innovation.

The open-source repository for ODS can be found at https://github.com/sentient-agi/OpenDeepSearch.
